VALIDATION_API int VO_delete_validationresult (tag_t vr_tag)
 

Detailed Description

Validation objects are created when targets are checked using validation agents. For example when a NX dataset is validated using NX Check-mate and the results are saved in to Teamcenter, the Validation results are saved in database. ValidationData objects represent the checkers and ValidationResult objects store the results in them.
