Detailed Description

The Instances of HSM_Policy are used in conjunction with 3rd Party Hierarchical Storage Management software(HSM) for migration of files between different volume tiers based on policies.

The policies are made up of Teamcenter Engineering metadata and various OS File specific attributes.

logical  fms_migration 
)

The function migrate all Teamcenter 2007 volume files under pending request. This function applies the migration related attribute changes on Teamcenter volume file object in database If this function is called in the context of HSM Migration than ensure that the physical files have been previously migrated by 3rd party HSM systems.

Parameters
request_tag(I) Migration Pending request object tag
fms_migration(I) True or False values depending upon FMS OR 3rd Party Migration
