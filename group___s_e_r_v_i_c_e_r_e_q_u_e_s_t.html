SERVICEREQUEST_API int SERVICEREQUEST_cancel_service_object ( tag_t  service_object)

Cancels the requested Service Request.

  • If the object is a Requested Activity or a Delegate Service Request object, the operation cancels it.
  • If the object is a Primary Service Request object, it cancels the Primary Service Request and all its Delegate Service Requests and its Requested Activities.
Parameters
service_object(I) The Service object to cancel

Substitutes an existing error in the error stack with a specified one.
If the error #error_to_replace is found in the error stack, its whole content is cleared and the new error #replacement_error is added to the error stack.

Returns
  • ITK_ok if the error #error_to_replace was found and the replacement was completed.
  • #replacement_error if the error #error_to_replace was not found in the error stack.
