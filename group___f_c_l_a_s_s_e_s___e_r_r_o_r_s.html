#define FCLASSES_mutex_unavailable   (EMH_FCLASSES_error_base + 28)

An error has occurred while acquiring the mutex for the segment "%1$-%2$". This is probably caused by insufficient system resources. Please report this to your system administrator.

NOTE: Because this error may occur within the text module, it is not possible to retrieve text in the current locale, without risking infinite recursion. Therefore, the error text is only ever in English (hard-coded in memorymanagement.cxx).
