SSF0SERVICEFORECASTING_API int SSF0SERVICEFORECASTING_cancel_maintenance_action ( int  num_cancel_maintenance_actions,
const tag_t *  cancel_maintenance_actions 
)

Cancels the Maintenance Action object.

Returns
  • ITK_ok on success.
  • SERVICEFORECASTING_ma_invalid_role error if logged in user role is not valid.
  • SERVICEFORECASTING_ma_in_approval error if maintenance action is in due date approval process.
Parameters
num_cancel_maintenance_actions(I) Number of maintenance actions to cancel
cancel_maintenance_actions(I) The maintenance actions to cancel

SSF0SERVICEFORECASTING_API int SSF0SERVICEFORECASTING_create_maintenance_action ( const SSF0SERVICEFORECASTING_create_maintenance_action_t *  maintenance_action_info,
tag_t *  created_maintenance_action 
)

Creates a Maintenance Action object.

Returns
  • ITK_ok on success.
  • SERVICEFORECASTING_invalid_due_date error if invalid date is provided.
  • SERVICEFORECASTING_ma_create_sc_absent error if structure context is not provided.
  • SERVICEFORECASTING_ma_create_sr_absent error if service requirement is not provided.
  • SERVICEFORECASTING_ma_create_physical_part_absent error if physical part is not provided.
Parameters
maintenance_action_info(I) Information to create a maintenance action objects.
created_maintenance_action(O) The created maintenance action

const tag_t *  extend_maintenance_actions,
date_t  request_date,
const char *  note 
)

Extends the due date for the Maintenance Action object.

Returns
  • ITK_ok on success.
  • SERVICEFORECASTING_invalid_due_date error if invalid date is provided.
  • SERVICEFORECASTING_duplicate_extension error if maintenance action is already in due date approval process.
Parameters
num_extend_maintenance_actions(I) Number of maintenance actions to be extended
extend_maintenance_actions(I) The maintenance actions to be extended
request_date(I) New proposed due date for the maintenance action
note(I) Note for extension object
