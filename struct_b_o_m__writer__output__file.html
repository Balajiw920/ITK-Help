Detailed Description

Output is sent to a filehandle. Example:

BOM_writer_output_file* p;
BOM_writer_new_output_file( &p );
p->filehandle = fopen("file", "r");
BOM_writer_write_bomwindow( p, ... );
fclose( p->filehandle );
BOM_writer_delete( p );
